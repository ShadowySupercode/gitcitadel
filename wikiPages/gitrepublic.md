# GitRepublic

GitRepublic is a [[[GitCitadel]] IDE that will integrate:
- a git controller featuring [[Ngit]] functionality, according to [[NIP-34]]
- a Nostr client
- documentation hosted on Nostr
- developer communication channels
- project management
